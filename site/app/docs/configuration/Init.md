#####initialScale:    
[integer] `0`  
Specifies the initial "height" of the zoom on the svg.   

#####initialTranslate:    
[2 element array] `[0,0]`  
Specifies the initial "pan" of the svg.

#####warningMessage: 
[string] `"There be no data!  What's going on?"`  
Specifies a custom warning message if there is no data.
